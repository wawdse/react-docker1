## 실습 명령어 

```sh
# 이미지 생성 - 이미지 이름: react-docker1
docker build -t react-docker1 -f Dockerfile.dev .

# 이미지 목록 보기
docker image ls

# 컨테이너 생성 및 실행
docker run -d --name react-docker1 -p 5173:5173 react-docker1

docker ps

docker exec -it 컨테이너ID bash

# 컨테이너 중지 및 제거
docker stop react-docker1
docker rm -f react-docker1

# 이미지 삭제
docker image rm [이미지ID]
```
